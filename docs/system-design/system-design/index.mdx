# System Design

系统设计是面试高级职位的必修项。区别于[面向对象程序设计](../index.mdx)，当我们在讨论系统设计时，我们更关注如何设计一个**大型、高可用、可扩展**的系统。

系统设计是一个综合性很强的话题，考虑的是对计算机科学领域的基础知识的综合应用。一个好的设计，从用户请求到存储，涉及到了很多方面，比如网络、存储、数据库、缓存、负载均衡、消息队列、搜索、安全等等。

不过虽然系统设计综合性很强，但是网络上有很多资源可以帮助我们学习，也有一些经典的题目与范式可以帮助我们更好地理解系统设计。

下面是一些系统设计的参考资源，本系列教程也是对这些内容的整合：
