# 基础知识

本章节会介绍计算机科学领域的一些基础知识，包括计算机网络、操作系统等等。虽然算法与数据结构也在计算机科学的基础知识范畴内，但是由于其重要性，单独在 [LeetCode](/docs/leetcode/re-leetcode) 章节专门介绍。

本章节的内容会面那些想要学习计算机行业基础知识的下人群：

1. CS 相关专业的学生
2. 软件行业从业者
3. 想要转行到软件行业的人

不论是**知识学习**，还是[**准备面试**](/docs/interview/index.mdx)，希望本章节的内容都能帮到你。
