# 关系型数据库

## 1. 常见开源、免费关系型数据库

当我们提及「数据库」时，通常指的是**关系型数据库（Relational Database）**。

这里列举一些常见的**开源、免费、关系型数据库**：

1. MySQL
2. MariaDB
3. PostgreSQL
4. SQLite

除了关系性数据库外，还有**非关系型数据库（NoSQL）**，我们将在 [NoSQL 数据库](../nosql/index.mdx) 中介绍。
